# Portfolio ASIR

## Presentación
Soy estudiante de ASIR y he creado este portfolio para mostrar proyectos prácticos de administración de sistemas, redes, virtualización, automatización, backups, monitorización y servicios de infraestructura.

## Objetivo profesional
Quiero demostrar habilidades reales en entornos Linux y Windows, así como en tareas de soporte, administración, despliegue y mantenimiento de sistemas.

## Habilidades
- Linux
- Windows Server
- Active Directory
- Virtualización
- Redes básicas
- Bash
- PowerShell
- Git y GitHub
- Copias de seguridad
- Monitorización
